Methods

Two-group grade of membership model

We start from Poisson factorization (PF):

\[ x_{ij} \sim \mathrm{Poisson}(\lambda_{ij}) \]

\[ \lambda_{ij} = \mathbf{LF} \]

The PF MLE can be converted to a grade of membership model by normalizing to satisfy the constraints:

\[ l_{ik} \geq 0 \]

\[ \sum_i l_{ik} = 1 \]

\[ f_{kj} \geq 0 \]

After normalizing, the interpretation of \(f_{ik}\) is the centroid of cluster \(k\) and \(l_{ik}\) is the cluster weight of sample \(i\) on cluster \(k\). In this model, sample \(i\) could potentially belong to all clusters.

For lineage tracing we propose instead to model sample \(i\) as belonging to at most two clusters. The interpretation is that sample loadings interpolate between reference points, which are factors. This corresponds to imposing the constraint:

\[ \sum_i [l_{ik} > 0] \leq 2 \]

We can solve the constrained optimization problem by introducing a Lagrange multiplier. The corresponding penalty weight \(\alpha\) needs to be pre-specified/tuned. 

import tensorflow as tf

def pois_llik(x, mean):
  return x * tf.log(mean) - mean - tf.lgamma(x + 1)

def gom2(x, k, alpha, size, F=None, learning_rate=1e-2, max_epochs=10000, verbose=True):
  n, p = x.shape
  graph = tf.Graph()
  with graph.as_default():
    x = tf.Variable(x, trainable=False)
    size = tf.Variable(size.reshape(-1, 1), trainable=False)

    loadings = tf.exp(tf.Variable(tf.random_normal([n, k])))
    if F is not None:
      factors = tf.Variable(F, trainable=False)
    else:
      factors = tf.exp(tf.Variable(tf.random_normal([k, p])))
    mean = size * tf.matmul(loadings, factors)
    llik = tf.reduce_sum(pois_llik(x, mean))
    penalty = alpha * tf.reduce_sum(tf.cast(loadings > 1e-4, tf.float32))
    loss = -llik + penalty

    train = tf.train.RMSPropOptimizer(learning_rate=learning_rate).minimize(loss)
    opt = [loadings, factors]
    obj = float('-inf')
    with tf.Session() as sess:
      sess.run(tf.global_variables_initializer())
      for i in range(max_epochs):
        _, update, pen = sess.run([train, loss, penalty])
        if not np.isfinite(update):
          raise tf.train.NanLossDuringTrainingError
        if verbose and not i % 500:
          print(i, update, pen, end='\r')
      if verbose:
        print(i, update)
      return sess.run(opt)

Plotting functions

Produce a STRUCTURE plot. 

def plot_structure(weights, ax=None, idx=None):
  if ax is None:
    ax = plt.gca()
  prop = np.cumsum(weights, axis=1)
  if idx is None:
    idx = np.lexsort(weights.T)
  for i in range(prop.shape[1]):
    if i > 0:
      bot = prop[idx,i - 1]
    else:
      bot = None
    ax.bar(np.arange(prop.shape[0]), weights[idx,i], bottom=bot, color=f'C{i}', width=1)
  ax.set_xlim(0, weights.shape[0])
  ax.set_ylim(0, 1)
